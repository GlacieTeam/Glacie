## 什么是Glacie？
Glacie是一个基于 `LeviLamina` Mod加载器的一个服务端Mod，让玩家可无缝连接到与客户端版本不同的服务器，无需担心版本不匹配导致的无法加入问题。

Glacie实际上是一个翻译器，试图通过翻译不同版本之间的数据包差异，来允许不同版本的玩家进入服务器。


## 安装

```bash
lip install github.com/GlacieTeam/Glacie
```
